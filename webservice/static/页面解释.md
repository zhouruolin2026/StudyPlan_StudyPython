## 逐行逐词解释

### `<!DOCTYPE html>`
- `<!` — 声明标签，不是普通 HTML 元素
- `DOCTYPE` — Document Type，文档类型
- `html` — 告诉浏览器这是 HTML5 文档
- 作用：让浏览器用标准模式渲染，不加会触发兼容模式

---

### `<html lang="zh-CN">`
- `<html>` — 根元素，整个页面的容器
- `lang="zh-CN"` — 声明页面语言是简体中文，辅助屏幕阅读器和搜索引擎

---

### `<head>` 区域

```html
<meta charset="UTF-8">
```
- `meta` — 元信息标签，不显示内容，给浏览器提供配置
- `charset` — character set，字符集
- `UTF-8` — 支持中文在内的几乎所有字符的编码方式

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
- `name="viewport"` — 控制手机端视口
- `width=device-width` — 页面宽度等于设备屏幕宽度
- `initial-scale=1.0` — 初始缩放比例为 1，不放大也不缩小

```html
<title>首页</title>
```
- 显示在浏览器标签页上的文字

---

### CSS 部分

```css
* { box-sizing: border-box; margin: 0; padding: 0; }
```
- `*` — 通配符，选中页面所有元素
- `box-sizing: border-box` — 让 width/height 包含 padding 和 border，否则加了 padding 会撑大元素
- `margin: 0` — 清除所有元素默认的外边距
- `padding: 0` — 清除所有元素默认的内边距

```css
body { font-family: Arial, sans-serif; background: #f0f2f5; height: 100vh; display: flex; flex-direction: column; }
```
- `font-family: Arial, sans-serif` — 优先用 Arial，没有则用系统默认无衬线字体
- `background: #f0f2f5` — 浅灰蓝背景色
- `height: 100vh` — 高度等于视口100%，撑满整个屏幕
- `display: flex` — 启用弹性布局
- `flex-direction: column` — 子元素竖向排列（默认是横向）

```css
.top-bar { padding: 12px 20px; background: white; box-shadow: 0 1px 4px rgba(0,0,0,0.1); display: flex; justify-content: space-between; align-items: center; }
```
- `.top-bar` — 选中 class 为 top-bar 的元素
- `padding: 12px 20px` — 上下12px、左右20px 的内边距
- `box-shadow: 0 1px 4px rgba(0,0,0,0.1)` — 阴影：水平偏移0、向下1px、模糊4px、黑色10%透明度
- `justify-content: space-between` — 子元素两端对齐，左边贴左、右边贴右
- `align-items: center` — 子元素垂直居中

```css
.site-title { font-size: 15px; font-weight: bold; color: #333; }
```
- `font-weight: bold` — 字体加粗
- `color: #333` — 深灰色文字

```css
.nav-links { display: flex; gap: 20px; }
```
- `gap: 20px` — flex 子元素之间的间距20px

```css
.top-bar a { color: #4f8ef7; text-decoration: none; font-size: 14px; }
.top-bar a:hover { text-decoration: underline; }
```
- `text-decoration: none` — 去掉链接默认下划线
- `:hover` — 鼠标悬停时的状态
- `text-decoration: underline` — 悬停时显示下划线

```css
.content { flex: 1; display: flex; flex-direction: column; align-items: center; justify-content: center; }
```
- `flex: 1` — 占据 body 中除 top-bar 以外的所有剩余高度
- `align-items: center` — 子元素水平居中
- `justify-content: center` — 子元素垂直居中

```css
.toolbar { width: 90%; display: flex; align-items: center; gap: 10px; margin-bottom: 10px; }
```
- `margin-bottom: 10px` — 工具栏与下方文本框之间留10px间距

```css
.toolbar span { font-size: 14px; color: #555; white-space: nowrap; }
```
- `white-space: nowrap` — 文字不换行，防止"跑批日期替换"被折断

```css
.toolbar input { padding: 8px 10px; border: 1px solid #ddd; border-radius: 6px; font-size: 14px; width: 160px; }
```
- `border: 1px solid #ddd` — 1像素实线浅灰边框

```css
.btn { padding: 8px 18px; border: none; border-radius: 6px; font-size: 14px; cursor: pointer; white-space: nowrap; }
```
- `border: none` — 去掉按钮默认边框
- `cursor: pointer` — 鼠标移上去变成手形，提示可点击

```css
.btn:disabled { background: #ccc; cursor: not-allowed; }
```
- `:disabled` — 按钮被禁用时的状态
- `cursor: not-allowed` — 鼠标变成禁止符号

```css
.toast { position: fixed; bottom: 30px; left: 50%; transform: translateX(-50%); background: rgba(0,0,0,0.7); color: white; padding: 8px 20px; border-radius: 20px; font-size: 13px; opacity: 0; transition: opacity 0.3s; pointer-events: none; }
.toast.show { opacity: 1; }
```
- `position: fixed` — 固定定位，不随页面滚动
- `bottom: 30px` — 距离窗口底部30px
- `left: 50%` — 左边缘在页面水平中心
- `transform: translateX(-50%)` — 向左偏移自身宽度的50%，实现水平居中
- `rgba(0,0,0,0.7)` — 黑色70%不透明
- `border-radius: 20px` — 圆角大到变成胶囊形状
- `opacity: 0` — 默认透明（不可见）
- `transition: opacity 0.3s` — opacity 变化时用0.3秒过渡动画
- `pointer-events: none` — 鼠标事件穿透，不会挡住下面的元素点击
- `.toast.show` — 同时有 toast 和 show 两个 class 时生效，把透明度改为1（可见）

### JavaScript 部分

```javascript
let lastSnapshot = null;
```
- `let` — 声明一个可变变量
- `lastSnapshot` — 变量名，存替换前的文本快照
- `null` — 空值，表示目前没有快照

```javascript
function showToast(msg) {
    const t = document.getElementById('toast');
    t.textContent = msg;
    t.classList.add('show');
    setTimeout(() => t.classList.remove('show'), 2000);
}
```
- `function` — 定义函数
- `msg` — 参数，调用时传入要显示的文字
- `const` — 声明常量，赋值后不能重新指向别的值
- `document.getElementById('toast')` — 在页面中找 id 为 toast 的元素
- `t.textContent = msg` — 设置元素的文字内容
- `t.classList.add('show')` — 给元素添加 show 这个 class，触发显示
- `setTimeout(() => ..., 2000)` — 2000毫秒（2秒）后执行箭头函数
- `() =>` — 箭头函数，简写的匿名函数
- `t.classList.remove('show')` — 移除 show class，触发隐藏

```javascript
function replaceDate() {
    const date = document.getElementById('replaceDate').value.trim();
    if (!date) { showToast('请先输入跑批日期'); return; }
    const ta = document.getElementById('mainText');
    if (!ta.value) { showToast('文本框内容为空'); return; }
    lastSnapshot = ta.value;
    document.getElementById('undoBtn').disabled = false;
    ta.value = ta.value.replaceAll('跑批日期', date);
    showToast('替换完成');
}
```
- `.value` — 获取输入框当前的文字内容
- `.trim()` — 去掉首尾空格
- `!date` — 感叹号是"非"，date 为空字符串时为 true
- `return;` — 提前结束函数，后面的代码不执行
- `lastSnapshot = ta.value` — 把当前内容存入快照变量
- `.disabled = false` — 解除撤回按钮的禁用状态
- `.replaceAll('跑批日期', date)` — 把文本中所有"跑批日期"替换成 date 的值

```javascript
function copyText() {
    const ta = document.getElementById('mainText');
    if (!ta.value) { showToast('文本框内容为空'); return; }
    if (navigator.clipboard && window.isSecureContext) {
        navigator.clipboard.writeText(ta.value)
            .then(() => showToast('已复制到剪切板'))
            .catch(() => showToast('复制失败，请手动复制'));
    } else {
        ta.select();
        ta.setSelectionRange(0, 99999);
        const ok = document.execCommand('copy');
        window.getSelection()?.removeAllRanges();
        showToast(ok ? '已复制到剪切板' : '复制失败，请手动复制');
    }
}
```
- `navigator.clipboard` — 浏览器剪切板 API 对象
- `window.isSecureContext` — 判断当前是否在安全上下文（HTTPS 或 localhost）
- `&&` — 逻辑与，两个条件都为真才进入 if
- `.writeText(ta.value)` — 把文字写入剪切板，返回一个 Promise
- `.then(...)` — Promise 成功时执行
- `.catch(...)` — Promise 失败时执行
- `ta.select()` — 选中 textarea 的全部文字
- `ta.setSelectionRange(0, 99999)` — 兼容移动端的全选写法，从第0个字符选到第99999个
- `document.execCommand('copy')` — 老式复制命令，返回 true/false 表示是否成功
- `window.getSelection()?.removeAllRanges()` — 取消文字选中状态；`?.` 是可选链，getSelection 为 null 时不报错
- `ok ? '已复制' : '复制失败'` — 三元运算符，ok 为 true 取前者，false 取后者

```javascript
function undoReplace() {
    if (lastSnapshot === null) return;
    const ta = document.getElementById('mainText');
    ta.value = lastSnapshot;
    lastSnapshot = null;
    document.getElementById('undoBtn').disabled = true;
    showToast('已撤回上次替换');
}
```
- `=== null` — 严格等于 null（三个等号，不做类型转换）
- `ta.value = lastSnapshot` — 把快照内容恢复到文本框
- `.disabled = true` — 重新禁用撤回按钮
